Automated Pothole & Road Surface Condition Monitor

Project Overview

A classical computer vision system to detect road defects using RGB camera input.

Pipeline

  1. Image Acquisition
  2. Preprocessing
  3. Homography (Bird’s-Eye View)
  4. LBP Feature Extraction
  5. Segmentation & Contour Detection
  6. Severity Grading

Severity Levels

Damage RatioSeverity
< 5%Minor
5–15%Moderate
> 15%Severe

Tech Stack: Python, OpenCV, NumPy, scikit-image
